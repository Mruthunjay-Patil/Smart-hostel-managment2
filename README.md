# Smart Hostel Management System

## Project Overview
Smart Hostel Management System is a Java-based desktop application developed using Java Swing and MySQL. The system helps manage hostel operations such as student registration, room allocation, fee management, and record maintenance.

## Features
- Student Registration
- Room Allocation
- Student Details Management
- Delete Student Records
- User-Friendly GUI using Java Swing

## Technologies Used
- Java
- Java Swing
- JDBC
- MySQL
- Eclipse IDE

## How to Run

1. Open the project in Eclipse.
2. Configure MySQL database.
3. Update database credentials in `DBConnection.java`.
4. Run the application.
5. Use the GUI to manage hostel records.
